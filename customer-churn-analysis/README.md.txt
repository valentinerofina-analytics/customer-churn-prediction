# Customer Churn Analysis & Prediction

## Project Overview
This project analyzes customer churn using Power BI and predicts churn using Python Machine Learning.

## Tools Used
- Power BI
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Dataset
Telecom Customer Churn Dataset containing customer demographics, services, billing, and churn status.

## Power BI Dashboard Insights
- Churn rate is approximately 26.5%
- Month-to-month contract customers churn the most
- Customers with high monthly charges show higher churn risk
- Low tenure customers churn more frequently

## Machine Learning Model
Model Used:
- Logistic Regression

Evaluation Metrics:
- Accuracy
- Recall
- ROC-AUC
